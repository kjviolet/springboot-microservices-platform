
# Spring Boot Microservices Platform

Backend platform built with Java and Spring Boot, designed with a microservices
approach and cloud-ready architecture.

## Tech Stack
- Java
- Spring Boot
- REST APIs
- SQL
- Docker
- Azure (cloud-ready)

## Architecture
- Microservices-based design
- Layered architecture (controller, service, repository)
- Externalized configuration

## Current Status
🚧 In progress: User Service (CRUD)

## Goals
- Demonstrate modern backend development practices
- Apply microservices and containerization concepts
- Serve as a portfolio project for enterprise backend roles
